This project is a platform for artists, intending to create projects with new people. Whether you want to make a band, shoot a music video, or need a writer, you can create the project and review people who may have the qualified skills for what you need.

This project is still in development. The backend is designed with Node.js, the frontend with Next.js, and the database with MongoDB.

Vercel link: https://music-plug.vercel.app/
